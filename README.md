# Urdu-Fake-News-Detection-NLP


## Project Overview
This project aims to detect fake news in the Urdu language using Natural Language Processing (NLP) techniques and machine learning models. The system classifies Urdu news articles as real or fake based on textual patterns and is more accurate than  https://abbdm.com/index.php/Journal/article/view/105/ 

## Dataset
- File: `urdu_news_complete_dataset.csv`
- Description: Contains labeled Urdu news articles used for training and testing the model.

## Methodology
- Data preprocessing and cleaning
- Tokenization using SentencePiece
- Feature extraction using NLP techniques
- Model training and evaluation

## Files in This Repository
- `NLP_project_Fake_URDU_News_Detect.ipynb` – Main Jupyter Notebook
- `urdu_news_complete_dataset.csv` – Dataset
- `config.json` – Model configuration
- `tokenizer.json` – Tokenizer file
- `tokenizer_config.json` – Tokenizer settings
- `special_tokens_map.json` – Special tokens
- `sentencepiece.bpe.model` – SentencePiece model
- `training_args.bin` – Training arguments

## How to Run the Project
1. Open the notebook in Jupyter Notebook or Google Colab
2. Install required Python libraries
3. Run all cells sequentially
